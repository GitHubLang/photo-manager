# 摄影素材管理系统

基于 AI 的摄影素材管理、Web 图片浏览、评分和文案生成系统。

---

## 功能特性

| 功能 | 说明 |
|------|------|
| 📁 目录树浏览 | 按日期整理素材，支持搜索 |
| 🔍 全局搜索 | 搜索文件名、描述、标签 |
| ⭐ AI 评分 | 本地 LLM / MiniMax 双模型支持 |
| 🏷️ AI 描述生成 | 自动生成图片描述和标签 |
| ✨ 相似图片检测 | 基于感知哈希去重 |
| 📝 每日主题总结 | AI 分析当日摄影主题 |
| 📱 抖音/小红书文案 | 一键生成可发布文案 |
| 🎯 优质图片推荐 | AI 筛选最佳图片组合 |

## 数据库设计

```sql
-- 图片基础信息
CREATE TABLE images (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    file_path VARCHAR(500) NOT NULL UNIQUE,
    filename VARCHAR(255) NOT NULL,
    folder_date DATE,
    folder_path VARCHAR(500),
    file_size BIGINT,
    width INT DEFAULT 0,
    height INT DEFAULT 0,
    orientation ENUM('landscape', 'portrait', 'square'),
    perceptual_hash VARCHAR(64),    -- 感知哈希，用于去重
    theme_tags VARCHAR(255),        -- 主题标签
    created_at DATETIME,
    indexed_at DATETIME,
    INDEX idx_folder_date (folder_date)
);

-- 图片评分
CREATE TABLE image_scores (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    image_id BIGINT NOT NULL,
    total_score DECIMAL(5,2),
    impact_score DECIMAL(5,2),      -- 冲击力 25%
    composition_score DECIMAL(5,2),  -- 构图 25%
    sharpness_score DECIMAL(5,2),    -- 清晰度 20%
    exposure_score DECIMAL(5,2),    -- 曝光 15%
    color_score DECIMAL(5,2),        -- 色彩 10%
    uniqueness_score DECIMAL(5,2),   -- 独特性 5%
    raw_response JSON,
    llm_model VARCHAR(100),
    scored_at DATETIME,
    FOREIGN KEY (image_id) REFERENCES images(id) ON DELETE CASCADE,
    INDEX idx_total_score (total_score)
);

-- 图片描述
CREATE TABLE image_descriptions (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    image_id BIGINT NOT NULL UNIQUE,
    description TEXT,
    tags VARCHAR(500),
    llm_model VARCHAR(100),
    created_at DATETIME,
    FOREIGN KEY (image_id) REFERENCES images(id) ON DELETE CASCADE
);

-- 每日主题
CREATE TABLE daily_themes (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    date DATE UNIQUE,
    theme_title VARCHAR(200),
    theme_description TEXT,
    photo_count INT,
    total_score_avg DECIMAL(5,2),
    keywords VARCHAR(500),
    created_at DATETIME
);

-- 推荐组合
CREATE TABLE photo_sets (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    date DATE,
    set_type ENUM('douyin', 'xiaohongshu', 'weibo'),
    cover_image_id BIGINT,
    caption_title VARCHAR(200),
    caption_body TEXT,
    hashtags VARCHAR(500),
    image_ids JSON,
    created_at DATETIME,
    FOREIGN KEY (cover_image_id) REFERENCES images(id) ON DELETE SET NULL
);
```

## AI 评分维度

| 维度 | 权重 | 说明 |
|------|------|------|
| 冲击力 (Impact) | 25% | 情感反应、视觉震撼力 |
| 构图 (Composition) | 25% | 三分法、黄金分割、平衡 |
| 清晰度 (Sharpness) | 20% | 对焦清晰、无模糊 |
| 曝光 (Exposure) | 15% | 无过曝/欠曝 |
| 色彩 (Color) | 10% | 白平衡、饱和度 |
| 独特性 (Uniqueness) | 5% | 原创性、独特视角 |

## 核心 API

| 接口 | 方法 | 说明 |
|------|------|------|
| `GET /api/folders` | GET | 获取目录树 |
| `GET /api/folders/{path}/images` | GET | 获取图片列表 |
| `POST /api/folders/scan-all` | POST | 扫描所有文件夹 |
| `POST /api/images/score` | POST | 批量评分 |
| `GET /api/search` | GET | 全局搜索 |
| `POST /api/daily-theme/{date}/generate` | POST | 生成主题 |
| `POST /api/caption/generate` | POST | 生成文案 |
| `POST /api/daily-report/{date}` | POST | 一键生成完整报告 |

## 使用流程

```
1. 启动后端服务
2. 打开前端页面 http://localhost:5173
3. 点击「扫描」导入所有图片到数据库
4. 选择一个日期文件夹浏览图片
5. 选择图片 → 点击「批量评分」
6. 评分完成后 → 点击「生成主题」
7. 选择喜欢的图片 → 点击「生成文案」
```
